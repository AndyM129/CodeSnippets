## Installation

可在任意路径下，通过终端执行以下命令，均会将代码片段安装该库到 Xcode 的目录下， 重启后即可生效使用：

```shell
curl -fsSL https://raw.githubusercontent.com/AndyM129/CodeSnippets/master/install.sh | sh
```
